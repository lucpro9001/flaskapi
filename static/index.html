Endpoint Method Description
/ GET Return infomation page
[PAYLOAD]: None
[RETURN]:
  • 200: a hmtl page
  • 4xx: some mistake from client
  • 5xx: server error
/payload GET GET payload category
[PAYLOAD]: None
[RETURN]:
  • 200: a json file contain payload info
  • 4xx: some mistake from client
  • 5xx: server error
api/predict/csv GET Take csv file to make prediction
[PAYLOAD]: file csv contain 13 feature column and a list of record needs to predict
[RETURN]:
  • 200: a csv file with additional predicted column indicate result
  • 4xx: file may be incorrect or some mistake from client
  • 5xx: server error
api/predict/csv POST Take csv file with 'true' lable(target) to stored and make model in a future
[PAYLOAD]: file csv contain 13 feature, 1 target column and a list of record needs to stored
[RETURN]:
  • 201: CREATED
  • 4xx: file may be incorrect or some mistake from client
  • 5xx: server error
