restful api
heart disease prediction

Endpoint Method Description
/ GET RESPONSE infomation page
[PAYLOAD]: None
[RESPONSE]:
  • 200: a hmtl page
  • 4xx: some mistakes from client
  • 5xx: server error
/payload GET GET payload categories
[PAYLOAD]: None
[RESPONSE]:
  • 200: a json file contains payload info
  • 4xx: some mistake from client
  • 5xx: server error
api/predict/csv GET Take a csv file to make prediction
[PAYLOAD]: file csv contain 13 feature columns and a list of record need to predict
[RESPONSE]:
  • 200: a csv file with id, predicted columns indicates result
  • 4xx: file may be incorrect or some mistakes from client
  • 5xx: server error
api/predict/csv POST Take a csv file with 'true' label(target) to save and to make model in a future
[PAYLOAD]: file csv contain 13 features, 1 target column and a list of record need to stored
[RESPONSE]:
  • 201: CREATED
  • 4xx: file may be incorrect or some mistakes from client
  • 5xx: server error
